## Run Locally

Clone the project

```bash
  git clone  
```

Go to the project directory
cd frontend

Install dependencies for frontend 

```bash
  npm install
```

Start the server

```bash
  npm run start
```

cd backend/expense-tracker

Install dependencies for frontend 

```bash
  npm install
```

Start the server

```bash
  npm run start
```
